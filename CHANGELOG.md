# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-24

### Added
- Initial release of personal portfolio website
- Responsive design for desktop, tablet, and mobile
- Hero section with introduction and research focus
- Research focus cards highlighting key areas
- Selected publications section with ORBIT and SMP-DLI
- Professional experience timeline (Capital One, UIUC, NCSA)
- Education section with degrees and honors
- Technical skills categorized by domain
- Contact section with multiple communication methods
- Smooth scroll navigation with active link highlighting
- Animated card reveals on scroll
- Mobile-friendly hamburger menu
- Custom 404 error page
- SEO optimization with meta tags and sitemap
- Open Graph tags for social media sharing
- robots.txt for search engine crawlers
- Favicon (brain emoji)
- GitHub Actions workflow for validation
- MIT License
- Comprehensive documentation (README, DEPLOY, QUICKREF, START_HERE)
- Automated deployment script (deploy.sh)
- Package.json with helpful scripts
- Contributing guidelines

### Design Features
- Professional blue color scheme (#2563eb)
- Inter font family for clean typography
- CSS Grid and Flexbox layouts
- Smooth transitions and hover effects
- Gradient backgrounds
- Card-based UI components
- Timeline visualization for experience
- Tag system for skills and research topics

### Performance
- Vanilla JavaScript (no frameworks)
- Optimized CSS with minimal specificity
- Lazy loading support for images
- Fast loading times (<100KB total)
- Progressive enhancement

### Accessibility
- Semantic HTML5 markup
- ARIA labels where appropriate
- High contrast text
- Keyboard navigation support
- Screen reader friendly

### Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
